meiryoKeGothic.ttc

meiryoKeConsole.ttf

●説明
これらのフォントは、「メイリオ」を従来の「ＭＳ　Ｐゴシック」のフォント幅に改良したフォントである。見慣れたフォント間隔で美しいアウトラインを得ることが出来る。
コンソール等で使える等幅フォントのmeiryoKeConsole.ttfがおまけについている。

これらのフォントの使用を「強制はしない」ので、各自好みで使用して欲しい。

●注意点
これらのフォントを使用した場合、配布する際にフォントを埋め込まなければならず、何かと不便になる。
PDF等にして配布する際は問題にならないが、pptやdotのまま配布する際は注意すること。


●使い方
１．「コントロールパネル」-「フォント」に突っ込む
